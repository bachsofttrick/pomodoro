# Online Pomodoro Timer

According to [Wikipedia](https://en.wikipedia.org/wiki/Pomodoro_Technique), the Pomodoro Technique is a time management method that splits your time working on a task into multiple intervals, intercepted by short breaks. Each interval is known as a pomodoro, from the Italian word for tomato, after the tomato-shaped kitchen timer the creator of this technique used as a university student.

## Usage

Open [https://bachsofttrick.github.io/pomodoro](https://bachsofttrick.github.io/pomodoro) directly in a browser.

## Features

- Work and break phases with distinct background colors (blue for work, green for break)
- Alarm sound plays at the last 10 seconds of each work session
- Settings saved to `localStorage` under key `pomodoro-time` in `work/break` format
- Default: 25-minute work session, 5-minute break
